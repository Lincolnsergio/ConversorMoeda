Conversor de Moedas

Este projeto é um Conversor de Moedas feito em Java, que funciona pelo console. A aplicação permite que o usuário escolha diferentes opções de conversão entre moedas e receba o valor convertido com base em taxas atualizadas obtidas por uma API.

A ideia do projeto é praticar conceitos importantes de desenvolvimento em Java, como consumo de API, leitura de dados em JSON e interação com o usuário no terminal.

Objetivo

Criar um programa que permita converter valores entre diferentes moedas através de um menu no console. O sistema deve oferecer pelo menos 6 opções de conversão.

As taxas de câmbio não são fixas. Elas são buscadas em tempo real por meio de uma API, garantindo resultados mais atualizados.

O que foi praticado no projeto

Durante o desenvolvimento foram trabalhados alguns pontos importantes:

Configuração do ambiente Java

Criação da estrutura do projeto

Consumo de uma API de taxas de câmbio

Leitura e análise de dados em JSON

Filtro das moedas necessárias

Exibição dos resultados no console para o usuário

Organização das tarefas

Pronto pra iniciar: tarefas que ainda não começaram

Desenvolvendo: tarefas que estão sendo feitas no momento

Concluído: tarefas finalizadas
